# 3D Tiltable Business Card

A customizable 3D business card web application that allows users to edit text, change colors, upload a logo, and download the card as an image.

## Features

- **3D Tilt Effect**: The card tilts based on mouse movement for a dynamic 3D appearance.
- **Editable Text**: Click on name, title, email, phone, and website fields to edit them directly.
- **Color Customization**: Adjust the gradient background using three color pickers (start, middle, end).
- **Font Selection**: Choose from various fonts including Segoe UI, Montserrat, Arial, etc.
- **Logo Upload**: Upload and display a custom logo on the card.
- **Download**: Save the customized card as a PNG image.
- **Responsive Design**: Adapts to smaller screens.

## Usage

1. Open `index.html` in a web browser.
2. Hover over the card to see the 3D tilt effect.
3. Click on the text fields to edit your information.
4. Use the color circles to change the gradient colors.
5. Select a font from the dropdown.
6. Upload a logo using the file input.
7. Click "Download Card" to save the image.

## Technologies Used

- HTML5
- CSS3 (with 3D transforms and gradients)
- JavaScript (ES6+)
- html2canvas library for image generation

## File Structure

- `index.html`: Main HTML file
- `style.css`: Stylesheet for the application
- `script.js`: JavaScript functionality
- `README.md`: This file

## Browser Compatibility

Works in modern browsers that support CSS 3D transforms and ES6 JavaScript.

## Troubleshooting

- If the download doesn't work, ensure your browser allows pop-ups for file downloads.
- For best results, use a modern browser like Chrome, Firefox, or Edge.
- If the logo doesn't display, check that the uploaded file is a valid image format.
